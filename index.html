Quiz Project
Project Description :

This project is an interactive Quiz Application designed to test users' knowledge on various topics. The quiz presents multiple-choice questions, allowing users to select an answer and navigate through a series of 25 questions.

Features :
User-Friendly Interface :

Clean and simple design for a seamless quiz-taking experience.

Multiple-Choice Questions :

Users can select from four answer options per question.

Choice Checking :

When a user selects an option, the correct answer will be highlighted in green, while incorrect answers will be shown in red.

Progress Indicator :

Displays the current question number out of the total (e.g., "Question 1 of 25").

Navigation Button :

When User Will Click the 'Next' button It will proceed to the next question.

Responsive Design :

Works smoothly on different screen sizes.

Technologies Used :

HTML – For structuring the quiz layout.

CSS – For styling and enhancing UI/UX.

JavaScript – For dynamic functionality and handling user interactions.

Purpose :

This quiz application is ideal for educational purposes, online assessments, and self-learning.

Features :

A clean and organized layout with a modern design to enhance user experience.

Hotel Information :

Technologies Used :

HTML -> structuring the webpage.

CSS -> styling and layout design.

JavaScript > interactivity and navigation.

Hotel Image & Name :
